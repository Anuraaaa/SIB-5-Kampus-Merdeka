HTML 5, singkatan dari HyperText Markup Language versi 5, adalah bahasa markup yang digunakan untuk membuat dan mengatur konten web. HTML 5 adalah evolusi dari versi sebelumnya, HTML 4, dan telah menjadi standar web yang paling umum digunakan

CSS 3, atau Cascading Style Sheets versi 3, adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan presentasi elemen-elemen dalam dokumen HTML atau XML. Ini adalah perkembangan dari CSS 2.1 dan merupakan bagian penting dari desain web modern.

Bootstrap adalah kerangka kerja (framework) front-end sumber terbuka yang digunakan untuk membangun situs web dan aplikasi web dengan cepat dan secara efisien. Dikembangkan oleh Twitter, Bootstrap menyediakan kumpulan alat, gaya, dan komponen yang telah dirancang sebelumnya untuk membantu pengembang web dalam mengembangkan tampilan yang responsif, menarik, dan konsisten.

JavaScript adalah bahasa pemrograman yang sering digunakan dalam pengembangan web untuk memberikan interaktivitas, fungsionalitas, dan dinamika pada halaman web.
